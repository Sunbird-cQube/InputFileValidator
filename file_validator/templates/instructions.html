Instructions

Follow these steps to upload your file:

  1. Click the "Browse" button or drag and drop your file into the designated area.
  2. If the file is successfully uploaded, you will see the file details below.
  3. To remove the uploaded file, click the "Delete" icon.
  4. Click the "Upload" button to start the upload process.
  5. Make sure the first column contains name in string.
  6. Make sure the second column contains age in integer.
  7. Make sure the third column contains email in string.
  8. If any errors occur during the processing, they will be displayed above.
